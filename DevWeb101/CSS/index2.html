Œufs sur le plat

Une illustration très laide, mais libre de droits.

Ingrédients

  1. une noisette de beurre ;

    Si le beurre est salé, on salera moins les œufs ensuite.

  2. des œufs

    prendre des œufs raisonnablement frais

Préparation

  1. Faire chauffer la poêlle
  2. la graisser avec le beurre
  3. casser les deux œufs dans la poêlle
  4. retirer du feu quand la consistence souhaitée est atteinte
